Stock Project By Team2




Instructions for using our website.



User needs to give amount in dollars and the types of companies for which they want to buy stock as Input.


The amount has to be more than $5000.00 for this engine to work.


User can select one or two types (strategies) of companies.


There are 5 available Strategies for selecting companies :


1. Ethical Investing

2. Growth Investing

3. Index Investing

4. Quality Investing

5. Value Investing
